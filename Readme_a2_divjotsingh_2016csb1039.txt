Open the terminal and enter the following command:	g++ a2_Divjot_2016CSB1039.cpp -o output `pkg-config --cflags --libs opencv`
Choose the options according to your need.
The size of both the images undergoing morphing should be the same and the separate files must have the points that are to be mapped to each other during morphing.
Sample case has been attached and it includes 80 points.
